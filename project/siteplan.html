🎬 Movie Explorer - Site Plan

Site Name

Movie Explorer

This name was chosen because it clearly represents a platform where users can explore and discover movies easily

Site Purpose

The website provides a platform for users to browse a collection of movies, view details, and discover new films. It will include dynamic content loaded from a data source and interactive features such as modals and saved favorites.

Scenarios

  • What movies are available to explore and what are their ratings?
  • How can I view more details about a specific movie?

Color Scheme

#121212 (Dark Background) – Used for the main background

#e50914 (Red Accent) – Used for headings and highlights

#ffffff (White) – Used for text

Typography

Oswald – Used for headings

Roboto – Used for body text
